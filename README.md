# SmartTutor

This is a helping infrastructure for the tutors worldwide. It will help them to manage their students and schedule and so on.

## features
Student profile: You can view your student information in this card and customize it.
Schedule: You can also set diffrent schedule for each student.
Everyday Sessions: You can also review how many tutions you have today. 
Alarm: You will get notified 1 hour before each tution time.
Schedule conflict preventer: If any schedule conflict happens it will automatically show up.
